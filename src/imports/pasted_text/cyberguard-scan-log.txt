Le scan Shannon a echoue

Details

L'etape "Run Shannon Pentest" a termine avec une erreur (exit code 1). Cela signifie generalement que Shannon n'a pas pu se lancer correctement ou que l'API key est manquante.

Actions correctives

1.
Verifiez que le secret ANTHROPIC_API_KEY est bien configure dans Settings → Secrets → Actions du repo GitHub.

2.
Verifiez que la cle API Anthropic est valide et non expiree.

3.
Assurez-vous que le fichier .github/workflows/cyberguard-scan.yml utilise correctement ${{ secrets.ANTHROPIC_API_KEY }}.

4.
Verifiez que Shannon est correctement installe dans le workflow (pip install shannon-ai ou equivalent).

5.
Consultez les logs detailles du step echoue sur GitHub Actions.
